# Titanic Dataset - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Titanic dataset to understand passenger survival patterns and key factors affecting survival.

The goal of this project is to analyze the dataset, clean the data, and visualize relationships between different features such as age, gender, class, and survival.

This project was created using Python in Google Colab.

---

## 📂 Dataset Information

Dataset Used: Titanic Dataset

Features include:

* PassengerId
* Survived
* Pclass (Passenger Class)
* Name
* Sex
* Age
* SibSp (Siblings/Spouses)
* Parch (Parents/Children)
* Ticket
* Fare
* Cabin
* Embarked

## 📊 EDA Steps Performed

The following analysis steps were performed:

1. Data Loading
2. Data Cleaning

   * Handling Missing Values
   * Removing Irrelevant Columns
3. Data Exploration

   * Checking Data Types
   * Statistical Summary
4. Data Visualization

   * Survival Count Plot
   * Gender vs Survival
   * Passenger Class vs Survival
   * Age Distribution
   * Fare Distribution
   * Correlation Heatmap

---

## 📈 Key Insights

Some observations from the analysis:

* Female passengers had a higher survival rate than male passengers.
* Passengers in 1st class had a higher survival rate.
* Younger passengers showed better survival probability.
* Higher fare passengers had better survival chances.
